# Home Lab Setup – Multi-OS Pentesting Environment

This directory contains the documentation for my locally built penetration testing lab.  
The environment was designed and deployed on a single laptop using Oracle VirtualBox with a bridged network topology.

## Lab Overview
- **Hypervisor:** Oracle VirtualBox 7.x (Type-2)
- **Network Mode:** Bridged (Layer-2), VMs obtain IPs from the home router
- **Virtual Machines:**
  - Kali Linux (192.168.1.242) – Attack workstation
  - Windows 10 (192.168.56.1) – Simulated corporate endpoint
  - Metasploitable 2 (192.168.1.21) – Intentionally vulnerable target server

## What's Inside
- Lab architecture and network diagram
- Detailed VM specifications and resource allocation
- Operational readiness checks and safety controls
- Screenshots of all configurations
